Данное программное обеспечение (далее - ПО) полностью написано мною, dgdagadin@gmail.com, от проектировки до разворачивания на хостинге. Контент данного ПО взят (и берется) с сайта 4stor.ru. Данное ПО является некоммерческим и разработано для использования мною и лицами, которым я дам доступ. ПО построено по принципу SPA (single-page application - одностраничное web-приложение).
Программа делится на три большие части:
  • Модуль синхронизации данных с сайтом 4stor.ru;
  • Серверные API-функции;
  • Клиентский функционал.
Программа гарантированно работает в следующих браузерах:
  • Internet Explorer 9+;
  • Google Chrome;
  • Opera;
  • Mozilla Firefox.
В остальных (кроме ИЕ 8 и менее) с большой долей вероятности заработает, но это я не проверял.
Технологии, использованные при реализации ПО:
  • На сервере:
    • PHP 5.6
    • MySQL 5.6
  • На клиенте:
    • Backbone.js 1.3.3
    • MarionetteJS 2.4.7
    • RequireJS 2.2.0
    • jQuery 2.2.4
    • jQuery UI 1.11.4
    • Underscore.js 1.8.3
    • Chartist.js 0.10.0
